## Convenções

- `projetos/mercado`: projetos com stacks atuais de mercado (TypeScript, Python, Java, Go).
- `projetos/portfolio-v2`: versões profissionais dos projetos que usam localStorage e interações completas.
- `projetos/legado-v1`: versões simples antigas para histórico.
- arquivos `projeto-*.html` na raiz: compatibilidade de URL (redirecionam para `portfolio-v2`).
